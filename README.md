# Jack_Kotlin 仿苹果备忘录

使用Kotlin语言开发的Android仿苹果备忘录

功能含量包括：基础界面编辑，RecyclerView,adapter,数据库存储，自定义dialog,编辑器，侧滑删除等，均使用kotlin进行开发，代码注释清晰，流程清楚。适合初学者使用，欢迎star
